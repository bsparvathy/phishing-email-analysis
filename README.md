Phishing Email Analysis

Overview

This project analyzes a phishing campaign documented by the UC Berkeley Information Security Office. The purpose of this analysis is to identify phishing indicators, understand social engineering techniques, and assess the risks posed by malicious emails.

Objectives

* Identify phishing characteristics.
* Analyze social engineering tactics.
* Assess risks associated with credential theft.
* Document findings in a structured format.

Skills Demonstrated

* Phishing Detection
* Security Awareness
* Social Engineering Analysis
* Risk Assessment
* Technical Documentation
